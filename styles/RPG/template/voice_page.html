Voice Chat

Provided by RolePlayGateway

Come have a conversation with us.

If you've got a headset (or if you're curious and just want to listen in!), you're more than welcome to come talk with fellow roleplayers on our voice chat.

How to get connected:

  1. Download and install the TeamSpeak Client.

  2. Open TeamSpeak and click "Connection" in the top left hand corner.

  3. Click "Quick Connect", and fill in the following information:

    Server address: roleplaygateway.com
    Nickname: Your name here.

  4. Click "Connect": You're all done!

Some Pro Tips

  • Set "Push to talk" - it works much better. This can be found in "Settings" -> "Sound Input/Output Settings".
  • Do an "echo test" (in the same menu as above) to make sure everything works.
  • Wait a few extra seconds before talking! You don't want to interrupt anyone.
